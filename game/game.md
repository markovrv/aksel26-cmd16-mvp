# Выберем профессию — образовательная 3D-игра-симулятор

**Цель:** профориентационная игра для школьников 7–9 классов со знакомством со строительными профессиями через интерактивные 3D-сцены и задачи.

**Стек:** Three.js + Vite + Vanilla JS + GSAP + jsPDF

---

## Профессии

### 1. Прораб 👷 (оранжевый #FF6B35)
- **Сцена:** строительная площадка (вид изометрия, OrbitControls)
- **Задача 1 — Приёмка материалов:** накладная на кирпич (520/500 шт), кнопки Принять/Отказать/Проверить ещё раз
- **Задача 2 — Распределение бригад:** drag-and-drop 5 рабочих в зоны Фундамент (2 чел) и Стены (3 чел)
- **Задача 3 — Аварийная ситуация:** анкерные болты закончились, 3 варианта решения
- **Макс. баллов:** 55 (10+15+20+10 бонус)

### 2. Инженер-энергетик ⚡ (синий #2563EB)
- **Сцена:** диспетчерская (вид от первого лица, параллакс, Bloom-эффект)
- **Задача 1 — Снижение перегрузки:** клик по индикатору ПС №2, переключить нагрузку на ПС №3
- **Задача 2 — Распределение резерва 50 МВт:** чекбоксы, выбор приоритетных потребителей
- **Задача 3 — Авария на ЛЭП:** отключить завод, дать свет домам и школе
- **Макс. баллов:** 55

### 3. Инженер КК 🔬 (зелёный #16A34A)
- **Сцена:** строящееся здание (вид от первого лица, 9 зон на стене)
- **Условие разблокировки:** пройти Прораба **и** Энергетика (любые результаты)
- **Задача 1 — Осмотр стены:** найти 3 скрытых дефекта — трещина, широкий шов, вздутие (клик по зонам 3x3)
- **Задача 2 — Проверка документации:** сравнить акт с нормативом СНиП, найти 3 ошибки (диаметр, бетон, подпись КК)
- **Задача 3 — Звонок заказчика:** давление директора подписать акт приёмки раньше срока
- **Макс. баллов:** 55

## Игровой цикл

```
Загрузка → Экран входа (имя + аватар) → 
  Главное меню (3 стенда) →
    Выбор профессии → Интро (3D-превью) → 
      Игровая сцена → Задача 1 → Задача 2 → Задача 3 → 
        Результаты (счёт, навыки, достижения, конфетти) →
          PDF / Играть снова / Другая профессия
```

## HUD (Heads-Up Display)

Постоянный HTML-оверлей поверх 3D-канваса:

- **Верхняя панель:** Игрок, Профессия, Счёт, Задача N/3, Таймер
- **Прогресс задач:** 3 кружка справа (серый/жёлтый/зелёный/красный)
- **Подсказки:** автоматические нижние подсказки (исчезают через 5 сек)
- **ЧП-индикатор:** пульсирующий красный блок (появляется в задаче 3)

## 3D-сцены

### Общие элементы
- PCFSoftShadowMap, ACESFilmicToneMapping
- AmbientLight + DirectionalLight + HemisphereLight
- Орбитальные контролы (настраиваются под каждую сцену)

### Главное меню
- 3 стенда-карточки (BoxGeometry 1.5×2.5×0.15)
- Плавающая анимация (sin по Y)
- Hover-эффект (scale 1.08, emissiveIntensity 0.4)
- Фонарные столбы с пульсирующими лампами
- Облака (дрейфуют по X)
- Камера: изометрия (0, 8, 14)

### Стройплощадка (Прораб)
- Фундамент BoxGeometry 6×0.3×4
- 3 стены с кирпичным материалом (grow-анимация)
- Грузовик: кузов, кабина, 4 колеса, фары (въезд/выезд по Z)
- Склад: BoxGeometry 3×2×2, коническая крыша
- 5 рабочих (капсула+сфера+конус), дыхание sin
- Зоны-сетки (PlaneGeometry wireframe)
- Аварийный красный свет (задача 3)
- Камера: изометрия (10, 8, 10)

### Диспетчерская (Энергетик)
- Тёмная комната (пол, потолок, стены)
- Главный щиток с 3 подстанциями (сферы+лампы)
- 3 монитора с графиками на canvas-текстурах
- Схема электросети (LineSegments)
- Аварийная сирена (CylinderGeometry + PointLight)
- Bloom-эффект (strength 0.4, задача 3 → 0.8)
- Параллакс камеры (мышь → смещение ±0.3)
- Камера: от первого лица (0, 1.8, 5)

### Стройобъект (Инженер КК)
- Пол с GridHelper (шаг 0.5 м)
- 4 колонны по углам
- Кирпичная стена 4×3×0.3 с 9 зонами (3×3 невидимые плоскости)
- 3 скрытых дефекта: трещина (LineSegments), шов (PlaneGeometry), вздутие (SphereGeometry)
- Инструменты: планшет, рулетка, уровень (декор)
- Телефон на стене (мигает в задаче 3)
- Висячая лампа PointLight
- Камера: от первого лица (0, 1.7, 3), OrbitControls, без pan

## Разблокировка профессии Инженер КК

Автоматически при возврате в главное меню после прохождения обеих базовых профессий:

```js
gameState.allCompleted = ['foreman', 'engineer'].every(
  id => gameState.history.some(h => h.profession === id)
);
```

Визуал разблокировки: стенд из серого → зелёный, подсветка, убирается замок, тост «🔬 Новая профессия открыта!»
